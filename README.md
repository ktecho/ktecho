# Hi, I'm Luis Miguel García

**Senior Software Engineer** with 25+ years building backends, infrastructure, and distributed systems — from the ground up.

## About

I ship production software across cloud infrastructure, DBaaS platforms, and distributed, high-availability systems. I've architected a multi-tenant cloud product on AWS/GCP (Terraform, Packer), led the migration of a high-traffic marketplace to a decentralized architecture, and built automated verification systems that validate shipped binaries against their source code. Along the way I founded a consulting firm, led engineering teams of up to 6, and delivered software for organizations from the National Bank of Spain to fast-moving startups.
